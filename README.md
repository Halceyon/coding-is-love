## 🛠 Installation

1. Clone the repo
2. Install NPM packages
```js
pnpm install
```
3. Run the project
```js
pnpm run dev
```

## 🌟 Features

- **GitHub Issues**: Track and manage issues directly from your landing page.

- **Microsoft ToDo tasks**: Sync your Microsoft ToDo tasks directly to your project.
